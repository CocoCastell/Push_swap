# About Push_swap

This algorithmic project is about sorting data with only 2 stacks and using the minimum of a limited set of instructions.

# Instructions

| Name | Purpose |
|------|---------|
|   sa | swap a |
|   sb | swap b |
|   ss | swap a & b |
|   pa | push a  |
|   pb | push b  |
|   ra | rotate a |
|   rb | rotate b |
|   rr | rotate a & b |
|  rra | reverse rotate a |
|  rrb | reverse rotate b |
|  rrr | reverse rotate a & b |
